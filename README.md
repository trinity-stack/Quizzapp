# QuizzApp
QuizzApp is a client-side quiz application that allows users to test their knowledge on computer programming.

## Features
- User accesses with username.
- Score tracking with localStorage.
- History for current user and global results.

## Technologies Used
- HTML
- CSS
- JavaScript 
- LocalStorage for data persistence

## How to Use
1. Open `index.html` in a web browser.
2. Enter your name to start the quiz.
3. Select answers to each question.
4. View your score and history on the results page.
5. Save your results to localStorage to track history.
